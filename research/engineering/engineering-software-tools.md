## Computer-Aided Design (CAD)

### AutoCAD (Autodesk)
- **Website:** https://www.autodesk.com/products/autocad/overview
- **Developer:** Autodesk
- **Pricing:** 
  - Commercial: ~$1,775/year
  - Student/Educator: Free (1-year renewable license)
- **Platforms:** Windows, macOS, Web, Mobile (iOS/Android)
- **Use Cases:** 2D drafting, architectural design, mechanical engineering, electrical design
- **Key Features:** Industry-standard DWG format, extensive toolsets (Architecture, Mechanical, Electrical), automation with AutoLISP

### SolidWorks (Dassault)
- **Website:** https://www.solidworks.com
- **Developer:** Dassault Systèmes
- **Pricing:**
  - Commercial: ~$3,995 (perpetual) + $1,295/year maintenance
  - Student: $99/year (many institutions provide free access)
- **Platforms:** Windows only
- **Use Cases:** Mechanical design, product development, simulation, 3D modeling
- **Key Features:** Parametric modeling, assembly design, integrated simulation, large community

### CATIA (Dassault)
- **Website:** https://www.3ds.com/products/catia
- **Developer:** Dassault Systèmes
- **Pricing:** 
  - Commercial: ~$12,000+ base license + $2,000/year maintenance (quote-based)
  - Student: Educational licenses available through institutions
- **Platforms:** Windows, Unix/Linux
- **Use Cases:** Aerospace, automotive, complex surface design, industrial design
- **Key Features:** Advanced surfacing, generative shape design, systems engineering, digital mockup

### Siemens NX
- **Website:** https://www.plm.automation.siemens.com/global/en/products/nx/
- **Developer:** Siemens Digital Industries Software
- **Pricing:**
  - Subscription: ~$287/month or ~$3,444/year
  - Quote-based for enterprise
- **Platforms:** Windows, Linux
- **Use Cases:** Advanced manufacturing, aerospace, automotive, product design
- **Key Features:** Convergent modeling, generative design, integrated CAM, synchronous technology

### PTC Creo
- **Website:** https://www.ptc.com/en/products/creo
- **Developer:** PTC (Parametric Technology Corporation)
- **Pricing:**
  - Subscription: $2,200 - $20,000/year (depending on modules)
  - Free 30-day trial available
- **Platforms:** Windows only
- **Use Cases:** Product design, mechanical engineering, generative design, simulation
- **Key Features:** Parametric and direct modeling, real-time simulation, AR integration, additive manufacturing

### Fusion 360 (Autodesk)
- **Website:** https://www.autodesk.com/products/fusion-360/overview
- **Developer:** Autodesk
- **Pricing:**
  - Commercial: $495/year
  - Startup/Hobbyist: Free (revenue <$100k/year)
  - Student/Educator: Free
- **Platforms:** Windows, macOS, Web browser
- **Use Cases:** Product design, CAD/CAM/CAE, electronics design, collaboration
- **Key Features:** Cloud-based, integrated CAD/CAM/CAE, generative design, PCB design, version control

### Onshape
- **Website:** https://www.onshape.com
- **Developer:** PTC (acquired from former SolidWorks founders)
- **Pricing:**
  - Professional: $1,500/year per user
  - Free: Limited public documents
  - Student/Educator: Free
- **Platforms:** Web browser (cloud-based)
- **Use Cases:** Mechanical design, real-time collaboration, product development
- **Key Features:** Full cloud CAD, real-time collaboration, version control, parametric modeling

### FreeCAD
- **Website:** https://www.freecadweb.org
- **Developer:** Open source community
- **Pricing:** Free (LGPL license)
- **Platforms:** Windows, macOS, Linux, Unix
- **Use Cases:** Parametric 3D modeling, mechanical engineering, architecture, 3D printing
- **Key Features:** Open source, parametric modeling, modular architecture, Python scripting, FEA workbench

### Tinkercad
- **Website:** https://www.tinkercad.com
- **Developer:** Autodesk
- **Pricing:** Free
- **Platforms:** Web browser
- **Use Cases:** Education, beginners, 3D printing, basic modeling
- **Key Features:** Block-based modeling, browser-based, integrated with Fusion 360, electronics circuits

### SketchUp

## Computational Analysis (CAE)

### ANSYS
- **Website:** https://www.ansys.com
- **Developer:** Ansys Inc.
- **Pricing:** Quote-based (typically $5,000-$50,000+ depending on modules)
- **Platforms:** Windows, Linux
- **Use Cases:** FEA, CFD, electromagnetic simulation, multiphysics
- **Key Features:** Industry-standard FEA, Fluent (CFD), comprehensive multiphysics, optimization tools

### Abaqus (Dassault)
- **Website:** https://www.3ds.com/products-services/simulia/products/abaqus/
- **Developer:** Dassault Systèmes (SIMULIA)
- **Pricing:** Quote-based (enterprise level)
- **Platforms:** Windows, Linux
- **Use Cases:** Advanced FEA, nonlinear analysis, composite materials, impact simulation
- **Key Features:** Robust nonlinear capabilities, explicit/implicit solvers, material modeling

### COMSOL Multiphysics
- **Website:** https://www.comsol.com
- **Developer:** COMSOL AB
- **Pricing:** Quote-based (~$4,000-$20,000+ depending on modules)
- **Platforms:** Windows, macOS, Linux
- **Use Cases:** Multiphysics simulation, custom PDEs, electromagnetics, heat transfer
- **Key Features:** Multiphysics coupling, equation-based modeling, application builder, CAD integration

### MATLAB/Simulink
- **Website:** https://www.mathworks.com
- **Developer:** MathWorks
- **Pricing:**
  - Individual: $149 (MATLAB) + toolboxes ($145-$2,000 each)
  - Student: $99 (MATLAB + add-ons bundle)
  - Home: $149
- **Platforms:** Windows, macOS, Linux
- **Use Cases:** Numerical computing, algorithm development, control systems, signal processing
- **Key Features:** Matrix-based computing, extensive toolboxes, code generation, visualization

### Wolfram Mathematica
- **Website:** https://www.wolfram.com/mathematica/
- **Developer:** Wolfram Research
- **Pricing:**
  - Standard: $1,595 (perpetual) or $595/year
  - Student: $225 or $75/year
- **Platforms:** Windows, macOS, Linux, Web, Mobile
- **Use Cases:** Symbolic computation, technical computing, data analysis, visualization
- **Key Features:** Symbolic and numeric computation, Wolfram Language, knowledgebase integration

### OpenFOAM
- **Website:** https://www.openfoam.com
- **Developer:** OpenCFD (ESI Group)
- **Pricing:** Free (GPL open source)
- **Platforms:** Linux (primary), Windows via WSL, macOS
- **Use Cases:** CFD, heat transfer, multiphase flow, reacting flows
- **Key Features:** Open source, highly customizable, parallel computing, extensive solver library

### CalculiX
- **Website:** http://www.calculix.de
- **Developer:** Open source community
- **Pricing:** Free (GPL license)
- **Platforms:** Windows, Linux, macOS
- **Use Cases:** FEA, structural analysis, thermal analysis, CFD
- **Key Features:** Open source, Abaqus-compatible syntax, implicit/explicit solvers

### Code_Aster
- **Website:** https://code-aster.org
- **Developer:** EDF (Électricité de France)
- **Pricing:** Free (GPL open source)
- **Platforms:** Linux, Windows (via WSL)
- **Use Cases:** Structural mechanics, thermal analysis, fatigue, seismic analysis
- **Key Features:** Open source, advanced material models, coupled physics, extensive validation
