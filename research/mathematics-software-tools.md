## 1. Computer Algebra Systems (CAS)

### Mathematica (Wolfram)
- **Website:** [https://www.wolfram.com/mathematica/](https://www.wolfram.com/mathematica/)
- **Pricing:** Commercial (from $1,080/year, Student: ~$170/year, Home & Hobby: $405)
- **Platforms:** Windows, macOS, Linux, Web (Wolfram Cloud)
- **Description:** Comprehensive technical computing system with symbolic computation, numerical analysis, data visualization, and machine learning capabilities.
- **Use Cases:** Research mathematics, physics simulations, engineering calculations, data science, education
- **Features:** Natural language input, notebook interface, extensive built-in knowledgebase, Wolfram Language

### Maple
- **Website:** [https://www.maplesoft.com/products/Maple/](https://www.maplesoft.com/products/Maple/)
- **Pricing:** Commercial (from $2,275 perpetual, Student: ~$129, Academic: ~$1,395)
- **Platforms:** Windows, macOS, Linux
- **Description:** Symbolic and numeric computing environment with emphasis on mathematics education and research.
- **Use Cases:** Calculus, algebra, differential equations, education, engineering
- **Features:** Clickable Math (point-and-click problem solving), comprehensive documentation, interactive components

### MATLAB
- **Website:** [https://www.mathworks.com/products/matlab.html](https://www.mathworks.com/products/matlab.html)
- **Pricing:** Commercial (from $940/year, Student: $49-$149, Academic: varies)
- **Platforms:** Windows, macOS, Linux, Web, Mobile
- **Description:** Numerical computing environment with strong focus on matrix operations, signal processing, and engineering applications.
- **Use Cases:** Engineering, signal processing, control systems, machine learning, image processing
- **Features:** Simulink integration, extensive toolbox ecosystem, Live Editor for interactive workflows

### SageMath
- **Website:** [https://www.sagemath.org/](https://www.sagemath.org/)
- **Pricing:** Free, Open Source (GPL license)
- **Platforms:** Windows (via WSL), macOS, Linux, Web (CoCalc)
- **Description:** Free open-source mathematics software system built on Python and numerous open-source packages.
- **Use Cases:** Number theory, cryptography, algebra, calculus, research mathematics
- **Features:** Combines NumPy, SciPy, SymPy, Matplotlib, R, and more; Jupyter notebook interface

### Maxima
- **Website:** [https://maxima.sourceforge.io/](https://maxima.sourceforge.io/)
- **Pricing:** Free, Open Source
- **Platforms:** Windows, macOS, Linux
- **Description:** Computer algebra system written in Common Lisp, based on DOE-MACSYMA.
- **Use Cases:** Symbolic computation, calculus, equation solving, matrix operations
- **Features:** Symbolic differentiation/integration, Taylor series, Laplace transforms, 2D/3D plotting

### SymPy
- **Website:** [https://www.sympy.org/](https://www.sympy.org/)
- **Pricing:** Free, Open Source (BSD license)
- **Platforms:** Cross-platform (Python library)
- **Description:** Pure Python library for symbolic mathematics.
- **Use Cases:** Symbolic computation integrated with Python workflows, education
- **Features:** Algebraic manipulation, calculus, equation solving, discrete math, quantum physics module
- **Installation:** `pip install sympy`

### PARI/GP
- **Website:** [https://pari.math.u-bordeaux.fr/](https://pari.math.u-bordeaux.fr/)
- **Pricing:** Free, Open Source (GPL license)
- **Platforms:** Windows, macOS, Linux
- **Description:** Specialized computer algebra system for fast computations in number theory.
- **Use Cases:** Number theory, algebraic geometry, cryptography research
- **Features:** Arbitrary precision arithmetic, factorization, elliptic curves, algebraic number theory

### Magma
- **Website:** [https://magma.maths.usyd.edu.au/](https://magma.maths.usyd.edu.au/)
- **Pricing:** Commercial (from $1,280/year, Academic: varies, Student: discounted)
- **Platforms:** Windows, macOS, Linux, Online Calculator
- **Description:** Computational algebra system designed for algebra, number theory, and geometry.
- **Use Cases:** Group theory, number theory, algebraic geometry, combinatorics
- **Features:** Advanced algorithms for algebraic structures, rigorous mathematical approach

## 2. Statistical Computing

### R
- **Website:** [https://www.r-project.org/](https://www.r-project.org/)
- **Pricing:** Free, Open Source (GNU GPL)
- **Platforms:** Windows, macOS, Linux
- **Description:** Programming language and environment for statistical computing and graphics.
- **Use Cases:** Statistical analysis, data visualization, bioinformatics, econometrics, research
- **Features:** Comprehensive statistical packages (CRAN), publication-quality graphics, reproducible research

### RStudio / Posit
- **Website:** [https://posit.co/products/open-source/rstudio/](https://posit.co/products/open-source/rstudio/)
- **Pricing:** Free (Desktop Open Source), Commercial (Workbench/Cloud from $1,045/year)
- **Platforms:** Windows, macOS, Linux, Server, Cloud
- **Description:** Integrated development environment (IDE) for R and Python.
- **Use Cases:** Data analysis, statistical programming, reproducible research
- **Features:** Syntax highlighting, debugging, R Markdown/Quarto support, Shiny app development, Git integration

### SPSS (IBM)
- **Website:** [https://www.ibm.com/spss](https://www.ibm.com/spss)
- **Pricing:** Commercial (from $1,800/year, Student: ~$100/year, Subscription available)
- **Platforms:** Windows, macOS, Linux (Server)
- **Description:** Statistical software package for social science research with point-and-click interface.
- **Use Cases:** Social sciences, market research, healthcare, government, education
- **Features:** Point-and-click interface, syntax programming, advanced statistics, data management

### SAS
- **Website:** [https://www.sas.com/](https://www.sas.com/)
- **Pricing:** Commercial (varies significantly, SAS University Edition: Free for learners)
- **Platforms:** Windows, macOS, Linux, Cloud
- **Description:** Comprehensive analytics platform for business intelligence and advanced analytics.
- **Use Cases:** Enterprise analytics, healthcare, finance, government, academic research
- **Features:** Advanced analytics, data management, business intelligence, machine learning

### Stata
- **Website:** [https://www.stata.com/](https://www.stata.com/)
- **Pricing:** Commercial (from $1,395 perpetual, Student: ~$225, Academic discounts available)
- **Platforms:** Windows, macOS, Linux
- **Description:** Complete statistical software package for data science and research.
- **Use Cases:** Economics, political science, sociology, biostatistics, epidemiology
- **Features:** Data management, statistical analysis, graphics, reproducible research, automated reporting

### JASP
- **Website:** [https://jasp-stats.org/](https://jasp-stats.org/)
- **Pricing:** Free, Open Source
- **Platforms:** Windows, macOS, Linux (including Apple Silicon)
- **Description:** User-friendly statistical software with both classical and Bayesian analysis.
- **Use Cases:** Psychology, social sciences, education, Bayesian statistics
- **Features:** Point-and-click interface, Bayesian analyses, integration with R, APA-style tables

### jamovi
- **Website:** [https://www.jamovi.org/](https://www.jamovi.org/)
- **Pricing:** Free, Open Source (AGPL3), Cloud subscription available
- **Platforms:** Windows, macOS, Linux, Cloud
- **Description:** "3rd generation" statistical spreadsheet designed for ease of use.
- **Use Cases:** SPSS alternative, teaching statistics, social sciences
- **Features:** Spreadsheet interface, R integration, module system for extensions, real-time analysis

## 3. Data Visualization & Analysis

### Python Ecosystem

#### NumPy
- **Website:** [https://numpy.org/](https://numpy.org/)
- **Pricing:** Free, Open Source
- **Description:** Fundamental package for numerical computing with Python; N-dimensional arrays, linear algebra, Fourier transforms.

#### SciPy
- **Website:** [https://scipy.org/](https://scipy.org/)
- **Pricing:** Free, Open Source
- **Description:** Scientific computing library for optimization, integration, interpolation, eigenvalue problems, signal processing.

#### Matplotlib
- **Website:** [https://matplotlib.org/](https://matplotlib.org/)
- **Pricing:** Free, Open Source
- **Description:** Comprehensive 2D/3D plotting library producing publication-quality figures.

#### pandas
- **Website:** [https://pandas.pydata.org/](https://pandas.pydata.org/)
- **Pricing:** Free, Open Source
- **Description:** Data manipulation and analysis library; DataFrames for structured data.

#### Jupyter
- **Website:** [https://jupyter.org/](https://jupyter.org/)
- **Pricing:** Free, Open Source
- **Platforms:** Cross-platform, Web-based
- **Description:** Interactive computing environment supporting notebooks, code, visualizations, and narrative text.

### Julia
- **Website:** [https://julialang.org/](https://julialang.org/)
- **Pricing:** Free, Open Source (MIT license)
- **Platforms:** Windows, macOS, Linux
- **Description:** High-level, high-performance programming language designed specifically for technical computing.
- **Use Cases:** Scientific computing, numerical analysis, machine learning, differential equations
- **Features:** JIT compilation, multiple dispatch, C-like performance with Python-like syntax, extensive math libraries

### GNU Octave
- **Website:** [https://octave.org/](https://octave.org/)
- **Pricing:** Free, Open Source (GPL)
- **Platforms:** Windows, macOS, Linux, BSD
- **Description:** High-level programming language for numerical computations, primarily MATLAB compatible.
- **Use Cases:** MATLAB alternative, numerical analysis, education, engineering
- **Features:** MATLAB-compatible syntax, extensive mathematical functions, 2D/3D plotting

### Scilab
- **Website:** [https://www.scilab.org/](https://www.scilab.org/)
- **Pricing:** Free, Open Source (CeCILL license)
- **Platforms:** Windows, macOS, Linux
- **Description:** Numerical computational package for scientific and engineering applications.
- **Use Cases:** Signal processing, control systems, simulation, optimization
- **Features:** Hundreds of mathematical functions, Xcos (dynamic systems modeler), 2D/3D visualization

## 9. Databases & Resources

### OEIS (Online Encyclopedia of Integer Sequences)
- **Website:** [https://oeis.org/](https://oeis.org/)
- **Pricing:** Free
- **Description:** Database of integer sequences, widely used by researchers and mathematicians.
- **Use Cases:** Sequence identification, mathematical research, pattern discovery

### LMFDB (L-functions and Modular Forms Database)
- **Website:** [https://www.lmfdb.org/](https://www.lmfdb.org/)
- **Pricing:** Free
- **Description:** Extensive database of L-functions, modular forms, and related number-theoretic objects.
- **Use Cases:** Number theory research, verification of conjectures, mathematical exploration

### nLab
- **Website:** [https://ncatlab.org/](https://ncatlab.org/)
- **Pricing:** Free
- **Description:** Wiki-lab for collaborative work on category theory, mathematics, and physics.
- **Use Cases:** Category theory research, higher mathematics reference

### MathWorld
- **Website:** [https://mathworld.wolfram.com/](https://mathworld.wolfram.com/)
- **Pricing:** Free
- **Description:** Extensive online mathematics encyclopedia by Wolfram Research.
- **Features:** 13,000+ entries, continually updated, extensively illustrated

### DLMF (Digital Library of Mathematical Functions)
- **Website:** [https://dlmf.nist.gov/](https://dlmf.nist.gov/)
- **Pricing:** Free
- **Description:** Online successor to Abramowitz and Stegun's Handbook of Mathematical Functions.
