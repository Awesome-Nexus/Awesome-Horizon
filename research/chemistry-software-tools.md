## Computational Chemistry

### Gaussian
| | |
|:---|:---|
| **Website** | https://gaussian.com/ |
| **License** | Commercial |
| **Platforms** | Windows, Linux, macOS, Unix |

Gaussian is a general-purpose computational chemistry software package initially released in 1970 by John Pople and his research group. It is one of the most widely used programs in computational chemistry.

**Key Features:**
- State-of-the-art electronic structure modeling (Gaussian 16)
- Predicts energies, structures, spectra, and molecular properties
- Wide range of methods: HF, DFT, MP2, CCSD(T), etc.
- Vibrational analysis and thermochemistry
- NMR chemical shift calculations
- Excited state calculations
- ONIOM multi-layer calculations

---

### ORCA
| | |
|:---|:---|
| **Website** | https://www.faccts.de/orca/ |
| **License** | Free for academic use, Commercial licenses available |
| **Platforms** | Linux, Windows, macOS |

A powerful and versatile quantum chemistry software package, primarily developed by the group of Prof. Frank Neese. One of the fastest-growing quantum chemistry software packages with 100,000+ registered academic users worldwide.

**Key Features:**
- Density functional and correlated wave function-based methods
- DLPNO-CCSD(T) for accurate binding/reaction energies
- RIJCOSX acceleration (up to 100-fold speedup for some features)
- NEB-TS for transition state finding
- Built-in QM/MM and ONIOM modules
- Crystal-QMMM for solid-state calculations
- Native molecular dynamics module
- Excited state dynamics (ORCA_ESD)
- Python interface (OPI) available

## Additional Resources

### File Format Support
Most chemistry software supports common file formats including:
- **MOL/SDF** (MDL MOLFile/SDFile)
- **PDB** (Protein Data Bank)
- **XYZ** (Cartesian coordinates)
- **CIF** (Crystallographic Information File)
- **SMILES/SMARTS** (Simplified Molecular Input Line Entry System)
- **InChI/InChIKey** (IUPAC International Chemical Identifier)
- **CML** (Chemical Markup Language)
- **CDX/CDXML** (ChemDraw)
- **MRV** (Marvin Document)

- **Python**: RDKit, Open Babel, PyMOL, ChimeraX

- **C/C++**: Open Babel, RDKit
