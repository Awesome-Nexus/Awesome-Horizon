## File Naming Convention

All resource files follow **kebab-case** naming convention:
- `master-resources.md` - Main comprehensive resource collection
- `learning-resources.md` - Learning-focused resources
- `global-resources.md` - Global/International resources by country
- `global-resources-extended.md` - Extended global coverage
- `communities-research.md` - Communities and research institutions
- `design-resources.md` - Subject-specific resources
- `steam-resources.md` - K-12 STEAM resources
